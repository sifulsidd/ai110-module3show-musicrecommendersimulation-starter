name: Always add explanatory comments above edits
description: User wants a comment block above every code or data change explaining what was added/changed and why
type: feedback
---

Always include a comment (or comment block) directly above any addition or edit in a file, explaining what the change is and why it was made.

**Why:** User explicitly requested this so changes are self-documenting and easy to review in context.

**How to apply:** For every Edit or Write that adds/modifies content, insert a comment line(s) immediately above the changed block. Use the appropriate comment syntax for the file type (# for Python/CSV context, // for JS, etc.). Apply to all file types including CSV, Python, and config files.
